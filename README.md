# 💰 Crypto Portfolio Watchlist

This is a simple web app where users can check cryptocurrency prices and keep track of their favourite coins. The project was made to practice JavaScript, working with APIs, and building a responsive webpage.

## About the Project

The app fetches live crypto data using the **CoinGecko API** and shows it in a dashboard. Users can search coins, sort them based on different values, and add coins to a personal watchlist.

## Features

* Search coins by name or symbol
* Sort coins by price, market cap, or 24h change
* Filter gainers and losers
* Add or remove coins from a watchlist
* Watchlist saved using **localStorage**

## Technologies Used

* HTML
* CSS
* JavaScript
* CoinGecko API
* Git & GitHub

## How to Run

1. Clone the repository
2. Open the project folder
3. Open **index.html** in your browser
